# Mystira.Infrastructure.Teams

Microsoft Teams bot infrastructure package for the Mystira platform.

## Features

- Microsoft Bot Framework integration
- Teams bot service implementation

## Installation

```bash
dotnet add package Mystira.Infrastructure.Teams
```

## Usage

```csharp
services.AddMystiraTeamsBot(configuration);
```

## Dependencies

- `Mystira.Application` - Application layer contracts
