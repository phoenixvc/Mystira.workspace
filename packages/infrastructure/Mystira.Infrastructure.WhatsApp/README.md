# Mystira.Infrastructure.WhatsApp

WhatsApp messaging infrastructure package for the Mystira platform.

## Features

- Twilio WhatsApp integration
- Azure Communication Services WhatsApp integration

## Installation

```bash
dotnet add package Mystira.Infrastructure.WhatsApp
```

## Usage

```csharp
services.AddMystiraWhatsApp(configuration);
```

## Dependencies

- `Mystira.Application` - Application layer contracts
