# Mystira.Infrastructure.Discord

Discord bot infrastructure package for the Mystira platform.

## Features

- Discord.Net integration
- Bot hosted service management
- Slash command modules
- Health checks for bot connectivity

## Installation

```bash
dotnet add package Mystira.Infrastructure.Discord
```

## Usage

```csharp
services.AddMystiraDiscordBot(configuration);
```

## Dependencies

- `Mystira.Application` - Application layer contracts
