# Content Bundle Use Cases

This directory contains documentation for content bundle-related operations.

## Overview

Content bundles represent collections of scenarios that can be purchased together. Bundle operations are primarily handled through services.

## Use Cases

Content bundle operations are currently handled through services:

- Content bundle CRUD operations (via services)
- Bundle pricing management (via services)
- Bundle purchase and access control (via services)

## Related Components

- **Domain Models**: `ContentBundle`, `BundlePrice`
- **Services**: Bundle-related services
- **Integration**: Subscription and purchase systems
